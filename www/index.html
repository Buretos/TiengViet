Справка

Как пользоваться приложением

🏠 Главный экран и уровни

Сетка из 6 уровней: A0 · A1 · A2 · B1 · B2 · C1. У каждого свой цвет. Тап по уровню — список из 50 уроков.

Прогресс по урокам сохраняется автоматически: при просмотре и при полностью прослушанном авто-плее.

🎧 Урок: озвучка и сохранение

В шапке урока — два ряда контролов:

Плеер:

  • ▶ Авто-плей — последовательное проигрывание всех фраз урока
  • 🐢 / 🚶 / 🏃 — скорость TTS (0.5× / 1.0× / 1.6×)
  • На Android звук продолжает играть автоматически и при выключенном экране — пока не нажать «⏹ Стоп».

На каждой фразе:

  • 🔊 VIE — озвучить вьетнамский
  • 🔊 RU — озвучить русский
  • ☆ → ★ — сохранить в папку «Избранное»

Повторное нажатие 🔊 — стоп. Размер шрифта урока меняется кнопками A− / A+ в шапке.

▶ Авто-плей урока

Последовательность для каждой фразы:

  1. Проход 1: RU → 1.8 с → VI → 1.5 с → VI → 2 с
  2. Проход 2: только VI → 2 с

Внизу всплывает плавающий бар с прогрессом, кнопками ⏸ Пауза и ⏹ Стоп. Любое нажатие 🔊 на отдельной фразе останавливает авто-плей.

📇 Карточки и папки

Вкладка «Карточки»:

  • + Карточка — создать свою (vi/ru/заметка/папка). Свои помечены ✏️
  • + Папка — создать новую папку
  • На каждой папке: ▶ авто-плей, Учить флэшкарты, 🗑 удалить
  • Тап на карточку — меню перемещения в другую папку
  • 🗑 на карточке — удалить

В авто-плее папки добавлена кнопка 🔁 Повтор — зацикленное проигрывание для пассивного прослушивания.

🎓 Режим «Учить» (флэшкарты)
  1. На карте — русский перевод. Попробуйте вспомнить вьетнамский.
  2. Тап по карте — переворот, показывает vi-фразу + автоматическая озвучка.
  3. Кнопки «Знаю» и «Не знаю».
  4. В конце сессии — повторение только «не знаю».
✏️ Письмо (Telex-ввод)

Тренировка набора фраз. Выбираете папку, нажимаете Начать практику — приложение даёт случайные фразы из неё.

Показывают русский перевод (с кнопкой 🔊 Послушать) — введите вьетнамский. Поле поддерживает Telex-ввод — даёт набирать вьетнамские буквы с обычной клавиатуры:

Сравнение проверяет точные диакритики и тоны (регистр и пробелы — нет). Кнопка 💡 Подсказка показывает первые символы.

🔍 Поиск

Поиск по всем 300 урокам. Фильтры: Всё / Вьетнамский / Русский.

Поле тоже поддерживает Telex — можно искать вьетнамское без вьетнамской раскладки. Тап на результат — открывает урок и прокручивает к нужной фразе.

📤📥 Синхронизация (экспорт / импорт)

На вкладке «Карточки»:

  • 📤 Экспорт — сохраняет все папки и карточки в JSON-файл
  • 📥 Импорт — загружает файл с другого устройства

На Android экспорт открывает системный шер — выбираете куда сохранить (Downloads / Drive / Telegram / e-mail). Копия также кладётся в Documents/ — найдёте через приложение «Files».

На Windows файл скачивается в стандартную папку Downloads.

При импорте появится диалог с выбором режима:

  • ➕ Дополнить — добавляет только отсутствующие папки и карточки. Существующие не трогаются.
  • ♻️ Заменить — стирает локальное и кладёт содержимое файла. С подтверждением.

Формат файла одинаков для Android и Windows — экспорт с одного устройства легко импортируется на другом.

💾 Где хранятся данные

Все ваши карточки, папки, прогресс и настройки лежат в localStorage приложения:

  • viet_cards — карточки
  • viet_folders — папки
  • viet_progress — прогресс по урокам

Базы Android и Windows независимы. Для синхронизации используйте 📤 Экспорт / 📥 Импорт.

Совет: делайте экспорт перед обновлением приложения или сменой устройства. Файл маленький (~1–10 KB).

❓ Частые вопросы

Первая фраза 🔊 запускается с задержкой.
TTS инициализирует системный движок. Следующие фразы — быстрее.

Импорт ничего не делает.
Файл должен быть JSON именно от этого приложения (с полем "app": "viet-learn").

Хочу очистить все карточки.
Импортируйте пустой файл в режиме ♻️ Заменить.
